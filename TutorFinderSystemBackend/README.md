## ðŸš€ Project Overview

The **Tutor Finder System Backend** allows:

- Students to search for tutors by subject, location, or rating  
- Tutors to manage their profiles, availability, and bookings  
- Admins to manage users, tutors, and bookings  

It provides a RESTful API for frontend applications to interact with the system.
